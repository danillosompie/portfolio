Intern Project


I was tasked with creating a key visual, designing key visual from scratch for various brand unit promotions. This request also included banners for ruparupa.com onsite, brand unit onsite, social media, newsletters, miniblocks, and more. (Slider banners, page banners, push notifications, search bar banners mobile apps and web version)


In the first couple images, I was tasked with creating a key visual that effectively highlights the promotion of the month. I designed the typographic logo by incorporating vectors elements such as a shopping cart and a gift box to enhance the promotional theme. For "It's Girls Time," I was tasked with creating a girly-themed key visual that is both appealing and eye-catching. I drew inspiration from the color palettes of animated cartoons for girls, incorporating pink as the dominant color and using heart shapes as visual decorative elements.

As you can see, in this brief, I was specifically asked to work on requirements for the brand unit Toys Kingdom. But I also did others brand unit, which is shown further below.



The last shown image above (Hotwheels), one of many designs,that is not a key visual I designed from scratch, but rather one based on a pre-existing key visual that I adapted and translated into various required media. A notable example is how I transformed the Hot Wheels packaging design into a pop-up banner format.
